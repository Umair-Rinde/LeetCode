3. Longest Substring Without Repeating Characters

Difficulty: Medium
Topics: String, Sliding Window, Hash Set
Companies: Multiple

Problem Statement:
Given a string s, find the length of the longest substring without duplicate characters.

Examples:

Example 1:
Input: s = "abcabcbb"
Output: 3
Explanation: The answer is "abc", which has a length of 3. Note that "bca" and "cab" are also valid substrings.

Example 2:
Input: s = "bbbbb"
Output: 1
Explanation: The answer is "b", with a length of 1.

Example 3:
Input: s = "pwwkew"
Output: 3
Explanation: The answer is "wke", with a length of 3.
Note: The answer must be a substring. "pwke" is a subsequence, not a substring.

Constraints:

* 0 ≤ s.length ≤ 5 × 10⁴
* s consists of English letters, digits, symbols, and spaces

Additional Info:

* This question has appeared in real interviews.
